Data Sources

Quicker Ticker works out of the box with no API keys required. These optional sources unlock more reliable data, higher refresh rates, and backup coverage if the default source has issues.

Yahoo Finance (default, no key needed)
Active
What's included

Real-time US stock prices, intraday and historical charts, company info, and news headlines. No account required. Note: Yahoo's data endpoints are unofficial and may occasionally be unavailable. Quicker Ticker uses a Stooq backup for historical data when Yahoo is unreachable.

Finnhub — Real-time quotes + news
Get a free key at finnhub.io — takes 30 seconds
What Finnhub adds

When entered, Finnhub becomes your primary price source with true real-time data. Falls back to Yahoo if Finnhub is unavailable. Finnhub free tier: 60 API calls/minute, US stocks, forex, and crypto. Does NOT include: international stocks outside US exchanges, detailed historical charts (those still use Yahoo).

Your API key is stored only in your browser's local storage. It is never transmitted anywhere except directly to api.finnhub.io.

Alpha Vantage — Fundamentals + earnings data
Get a free key at alphavantage.co
What Alpha Vantage adds

Unlocks company fundamentals (P/E ratio, EPS, revenue), earnings calendar, and sector data shown in expanded ticker view. Free tier: 25 requests/day, 5/minute — Quicker Ticker caches aggressively to stay within limits. Does NOT include: real-time prices (those stay on Yahoo/Finnhub) or news headlines.

Your API key is stored only in your browser's local storage. It is never transmitted anywhere except directly to alphavantage.co.

CoinGecko — Crypto prices + market data
What CoinGecko adds

More accurate crypto prices, market cap rankings, and 24h volume for coins not listed on Yahoo Finance. The free tier (no key) works but has rate limits. A free API key from coingecko.com raises those limits significantly.

Your API key is stored only in your browser's local storage. It is never transmitted anywhere except directly to api.coingecko.com.

Data & privacy

Quicker Ticker stores your watchlist and preferences in your browser only. If you use AI summaries, only the ticker symbol and its headlines are sent to the AI — never your browsing history, identity, or any other personal data. No analytics. No tracking pixels. No backend server.
